하네스 마인드맵 콘솔을 브라우저로 연다. **읽기 전용 실행 — 파일을 고치지 않는다.**

## 절차
1. 루트에 `registry.json` 이 없으면: 하네스 아님 → `/harness-init` 또는 `/harness-adopt` 안내하고 종료.
2. 루트에 `serve.py` 가 없으면: 구버전 골격 → `/harness-init` 재실행(공유 인프라 갱신) 안내하고 종료.
3. `serve.py` 를 **백그라운드**로 실행하고 출력을 임시 로그로 캡처:
   - `python serve.py` (인자 없음 = 8000 시도, 사용 중이면 빈 포트 자동 배정) 를 백그라운드로 띄운다.
   - 출력은 임시 로그로(예: `mktemp` / `%TEMP%` / 스크래치패드). 저장소를 더럽히지 않는다.
4. 로그에서 `http://localhost:<포트>/harness/` 한 줄을 읽는다(최대 ~2초). 안 나오면 로그를 그대로 보여주고 종료.
5. 그 URL 을 OS 기본 브라우저로 연다:
   - Windows: `cmd /c start "" "<URL>"` · macOS: `open "<URL>"` · Linux: `xdg-open "<URL>"`
6. 사용자에게 한 줄: 열린 URL + "백그라운드 실행 중(끄려면 그 포트의 파이썬 종료)".

## 주의
- 서버를 이미 띄워 둔 경우 새 인스턴스가 빈 포트로 또 뜬다(중복). 한 개만 쓰려면 기존 것을 끄고 다시 실행.
- 외부 노출 안 함 — 127.0.0.1 에만 바인딩.
- `python` 이 없으면 `python3` 로 재시도.
