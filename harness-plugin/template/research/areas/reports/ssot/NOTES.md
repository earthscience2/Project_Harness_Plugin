# 보고서

HTML 보고서를 관리하는 영역. 실파일은 [`reports/`](../../../reports/) 디렉터리.

- **목록 보기:** [`reports/index.html`](../../../reports/index.html) — 브라우저로 열면 갤러리.
- **새 보고서:** `reports/_template.html` 복사 → 내용 작성 → `reports/reports.json` 에 한 줄 추가.
- 보고서는 디자인시스템 스타일을 그대로 쓴다(일관된 룩).

스키마·절차는 [`reports/README.md`](../../../reports/README.md).
