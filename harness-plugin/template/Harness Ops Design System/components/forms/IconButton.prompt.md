**IconButton** — square icon-only button for toolbars and row actions; always pass `label` (used as aria-label + tooltip).

```jsx
<IconButton label="More actions" variant="ghost"><MoreIcon/></IconButton>
<IconButton label="Re-run" variant="subtle" size="sm"><RefreshIcon/></IconButton>
```

Same variants/sizes as Button. Default variant is `ghost`.
