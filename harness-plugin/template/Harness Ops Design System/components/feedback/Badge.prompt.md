**Badge** — compact label for counts, categories, versions. Tone conveys meaning, `variant` conveys emphasis.

```jsx
<Badge tone="accent">v2.4</Badge>
<Badge tone="success" variant="soft">통과</Badge>
<Badge tone="warning" variant="outline" pill>대기 12</Badge>
```

Tones: `neutral | accent | success | warning | danger | info`. Variants: `soft | solid | outline`. For run lifecycle states use **StatusBadge** instead.
